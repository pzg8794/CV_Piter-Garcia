# CV_Piter-Garcia

This repo now serves as the organized home for resume, CV, LinkedIn, and opportunity-specific professional materials.

## Structure

### Root resume sources
- `main.tex`: existing general academic/research CV source
- `main2.tex`: alternate resume/CV variant
- `main3.tex`: scholarship/leadership-oriented variant
- `ai_hub_resume.tex`: AI Hub targeted resume draft
- `ai_hub_resume.pdf`: compiled AI Hub draft

These remain at the repo root so the current Overleaf/Git workflow does not break.

### Materials
- `materials/ai_hub/`: AI Hub-specific planning, posting, and recommendation materials
- `materials/linkedin/`: LinkedIn source-of-truth documents and profile update packet
- `materials/scholarships/`: scholarship/fellowship-facing resume and essay artifacts

## Current Organization Rule

- Put opportunity-specific support materials in `materials/<target>/`
- Keep active `.tex` resume sources at the repo root unless you later decide to restructure the Overleaf build
- Prefer adding new target folders instead of scattering planning and wording files outside the repo

## Notes

- Files copied into `materials/` were imported from the surrounding scholarship workspace so they live with the resume system
- Nothing was deleted from the scholarship folder during this import
